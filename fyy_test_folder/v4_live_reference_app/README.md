# v4 实盘参照面板

一个放在 `fyy_test_folder` 下的独立静态页面，用来辅助执行 `v4_fast(best)`：

- 看今天是第几轮、轮次第几天
- 看今天属于选股日、买入日、持有日还是到期卖出日
- 记录当轮筛选名单
- 记录实际买价、最新价
- 自动计算 `-8%` 条件单参考价
- 提示今天应做的动作

## 打开方式

直接在浏览器打开：

- `fyy_test_folder/v4_live_reference_app/index.html`

不需要启动前端服务。

如果要直接导入你 2026-04-02 的首轮实盘状态，可使用：

- `fyy_test_folder/v4_live_reference_app/live_state_2026-04-02.json`

## 使用步骤

1. 先把真实交易日粘贴进页面。
2. 填写首轮选股日，点击“根据交易日生成轮次”。
3. 每轮把筛选结果录进去。
4. 买入后补上“实际买价”，页面会自动算出止损参考价。
5. 每天更新“今日日期”和“最新价”，看页面给出的今日动作。

## 数据保存

- 页面数据默认保存在浏览器 `localStorage`
- 也支持导出 / 导入 JSON

## 说明

- “快速生成工作日”只排除周末，不会自动排除 A 股节假日
- `live_state_2026-04-02.json` 里的交易日也只是工作日模板，导入后建议按真实 A 股交易日自行修正
- 实盘建议使用你自己的真实交易日列表
- 页面是执行参照工具，不替代最终交易判断
